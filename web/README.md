# FuelKenya Web

Frontend dashboard for the FuelKenya API.

## Local setup

Install dependencies with pnpm:

```bash
cd web
pnpm install
pnpm dev
```

If pnpm is not installed locally, install it first with:

```bash
corepack enable
corepack prepare pnpm@8.8.0 --activate
```

Then start the app and open `http://localhost:3000`.
